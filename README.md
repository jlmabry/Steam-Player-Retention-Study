# Steam Player Retention Analysis

## Overview
A data analysis project exploring which game attributes influence long-term player retention on Steam. Includes JSON parsing, data cleaning, and visualizations.

## Features
- Large-scale JSON dataset processing
- Exploratory data analysis (EDA)
- Visualizations of retention patterns
- Genre and tag-based comparisons

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- JSON Processing

## How It Works
Multiple JSON files were merged and cleaned, then analyzed to identify correlations between game attributes and player retention metrics.
